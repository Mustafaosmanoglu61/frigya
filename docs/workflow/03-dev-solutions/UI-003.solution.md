## 🎯 Ne yapacağız (1 cümle)
Lot tablosundaki "Toplam K/Z" ve "K/Z %" başlıklarına, bunların **gerçekleşmemiş (unrealized)** değerler olduğunu anlatan bir info ikonu + tooltip ekle.

## 🛠️ Yaklaşım (yön)

- `fiyatlar.html`'deki mevcut **`durum-info-ico`** pattern'ini birebir uygula:
  başlık metninin yanına `<i class="bi bi-info-circle ...">` + `title="..."`.
- Tooltip metni (her iki başlık için aynı):
  > "Gerçekleşmemiş (unrealized) — pozisyon hâlâ açık, satılmadı. Anlık fiyata göre hesaplanır."
- Native `title` attribute yeterli (mobil/touch'ta da çalışır, ekstra JS gerekmez).
- Ikon stilini mevcut CSS class'ından al (yeni renk/spacing icat etme); dark mode mevcut pattern'de zaten çözülü.

**Pseudo (th içeriği):**
```
<th ...>Toplam K/Z
   <i class="bi bi-info-circle <mevcut-ikon-class>"
      title="Gerçekleşmemiş (unrealized) — pozisyon hâlâ açık, ..."></i>
</th>
```

## ⚠️ Dikkat Et
- Sadece **görsel** değişiklik — hesap/JS'e dokunma.
- İkon başlık metninin **içinde** olsun ki kolon sürükle-bırakta birlikte taşınsın.
- `col-draggable` davranışını bozma (ikon tıklaması drag'ı tetiklememeli — pattern zaten bunu hallediyor, `event.stopPropagation` varsa koru).
- Dark mode kontrastını mevcut `durum-info-ico` dark kuralından devral.
